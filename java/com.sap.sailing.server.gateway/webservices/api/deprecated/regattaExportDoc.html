SAP Sailing Analytics - Web Services

Name: regatta

Description:

This service exports the structure (series, fleets, races) of a regatta


Webservice Type: REST
Output format: Json
Base-Url: http://www.sapsailing.com/sailingserver/regatta
Mandatory parameters: name
Optional parameters: None
Request method: GET
Example: http://tw2013.sapsailing.com/sailingserver/regatta?name=TW%202013%20%28Hobie%20Tiger%29
