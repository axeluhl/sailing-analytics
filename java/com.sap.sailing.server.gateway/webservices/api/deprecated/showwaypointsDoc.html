SAP Sailing Analytics - Web Services

Name: wind

Description:

This service lists the waypoints of a race's course and the mark positions at the time requested or at the end of the race if no time point is provided.


Webservice Type: REST
Output format: Json
Base-Url: http://www.sapsailing.com/sailingserver/moderator?action=showwaypoints
Mandatory parameters:
regattaname
racename
Optional parameters:
time
timeasmillis
Request method: GET
Example: http://www.sapsailing.com/sailingserver/moderator?action=showwaypoints&regattaname=ESS%202012%20Cardiff%20%28Extreme40%29&racename=Cardiff%20Race12
