SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/scopemigration/movetoarchiveserver

Description:

This endpoint is used to move a leaderboard group set from a dedicated server to an archive server.


Webservice Type: POST
Output format: json
Mandatory parameters:
archive, the host name of the archive server.
dedicated, the host name of the dedicated server (where the groups currently live).
UUID[], a set of leaderboardgroup UUIDs.
user1, username for archive server.
user2, username for dedicated server.
password1, password for user1 on archive server.
password2, password for user2 on dedicated server.
bearer1, bearer token for user1 on archive server, to provide instead of username and password.
bearer2, bearer token for user2 on dedicated server, see bearer1.
Request method: POST
Example: curl -X POST localhost:8889/sailingserver/api/v1/scopemigration/movetoarchiveserver -d "user1=admin" -d "user2=admin" -d "password2=admin" -d "password1=admin" -d "UUID[]=75e86618-8b0f-4291-9661-6b2a1fa47dff" -d "archive=localhost:8888" -d "dedicated=localhost:8889" -H "Authorization: Basic YWRtaW46YWRtaW4="
