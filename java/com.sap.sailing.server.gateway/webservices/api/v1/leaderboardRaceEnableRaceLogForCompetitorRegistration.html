SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/leaderboards/{leaderboardName}/enableRaceLogForCompetitorRegistration

Description:

This service will set a race to use racelog based competitor registrations


Webservice Type: REST
Output format: StatusCode
Mandatory parameters: raceColumnName
fleetName
raceName
Optional parameters: None
Request method: POST
Example: https://sapsailing.com/sailingserver/api/v1/leaderboards/test/enableRaceLogForCompetitorRegistration?raceColumnName=R2&fleetName=Default&raceName=R2
